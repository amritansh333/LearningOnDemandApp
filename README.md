# LearningOnDemand AI Chat App  An AI-powered tutor built with Python, Streamlit, and OpenAI.  ## Setup 1. Clone the repo: `git clone https://github.com` 2. Install dependencies: `pip install -r requirements.txt` 3. Add your OpenAI API Key to your environment variables. 4. Run the app: `streamlit run app.py`
